# Personal Payroll Hub 1.0

A clean payroll tracking app with no prefilled employee, company, joining date, payroll, or PDF data.

Included

- Blank employee and company profile
- Payroll entry, edit, delete, search, and filters
- Automatic net pay calculation
- Dashboard, reports, analytics, and pay calendar
- PDF pay stub vault
- Backup and restore
- Light, dark, and system themes
- Installable PWA
- Offline support
- Responsive desktop and mobile layout
- GitHub Pages ready

Run locally

1. Extract the ZIP.
2. Open the folder in VS Code.
3. Install the Live Server extension.
4. Right-click index.html.
5. Select Open with Live Server.

GitHub Pages

1. Upload all files and folders to a GitHub repository.
2. Open Settings, then Pages.
3. Select Deploy from a branch.
4. Choose main and root.
5. Save and wait for deployment.

First use

Open Profile & Settings and enter the employee name, company, joining date, and pay frequency. Then add the first paycheck.
